Casa de apuestas online

Idea

Crear una web donde se puedan crear apuestas de todo tipo y no solo deportivas como suele ser habitual.

Implementación

Para la creacion de apuestas existen varios modos posibles; manualmente y automaticamente.
La creacion de apuestas automaticas se podría hacer bien usando la API de espn para el seguimeinto de deportes o bien haciendo web scraping. La API de espn es de pago y hacer web scraping sería algo exagerado. Así que el sistema de creación de apuestas se ha mantenido bastante simple de modo que solo se permiten apuestas creadas manualmente donde las opciones posibles para apostar son listas y las participaciones en las apuestas son simples.

Cálculo de cuotas

La forma en la que he decidido calcular las cuotas es la europea (glosario de apuestas). En caso de que alguna opción no tenga participantes se indica.

Herramientas y recursos adicionales utilizados

  • Aptana como IDE para HTML, CSS, JavaScript y Python.
  • InkScape para crear las imágenes.
  • Firefox como navegador para probar el funcionamiento.
  • Firebug para retocar estilos y hacer debug de páginas.
  • Git para el control de versiones.
  • Twitter bootstrap como base para los estilos. Se ha utilizado para tener un reset de los estilos y una ayuda para posicionar los elementos con objetivo de hacer un diseño responsive.
  • jQuery como libreria JavaScript.

Notas sobre el funcionamiento y otras cuestiones

  • Usuario administrador inicial: admin | password: password
  • Usuario inicial: user1 | password: password
  • Al sincronizar la base de datos se crea el perfil para el admin en la aplicación y el usuario user1 además de algún ejemplo de apuesta.
  • Los usuarios normales no pueden crear apuestas.
  • Después de crear una apuesta hay que indicar que opción es la ganadora desde la administración de apuestas (botón "Administrar apuestas").
  • En el menú de administración de apuestas solo se muestran las apuestas pendientes de indicar la opción ganadora.
  • Se puede indicar la opción ganadora antes de acabar la apuesta, en ese caso la apuesta acaba inmediatamente.
  • El dinero de cada usuario se puede ver en su perfil.
  • En caso nadie apueste por una opción en concreto la cuota que aparece es 0.00, significa que en caso de apostar por esa opción te llevas todo el dinero.
  • Se puede apostar varias veces sobre una misma apuesta.
